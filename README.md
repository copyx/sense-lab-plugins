# Sense Lab Plugins

A personal Claude Code plugin marketplace by jingi.kim.

## Available Plugins

| Plugin | Description | Version |
|--------|-------------|---------|
| [english-proofreader](./english-proofreader) | Proofreads English prompts with educational feedback | 1.2.0 |

## Installation

### Step 1: Add Marketplace

```bash
/plugin marketplace add jingi-kim/sense-lab-plugins
```

### Step 2: Install Plugin

```bash
/plugin install english-proofreader@sense-lab-plugins
```

> **Note:** english-proofreader requires [Bun](https://bun.sh). Install with `curl -fsSL https://bun.sh/install | bash`
